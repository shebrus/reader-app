# 📚 Reader App

Мобильное приложение для чтения и озвучки книг.

## 🚀 Возможности

* 📖 Чтение книг (EPUB, TXT)
* 🎧 Генерация аудиокниг (TTS)
* 📝 Заметки и конспекты
* 🔄 Офлайн-доступ

## 🛠 Технологии

* React Native (Expo)
* TypeScript
* SQLite
* FastAPI (backend)

## 📱 Запуск проекта

```bash
npm install
npx expo start
```
