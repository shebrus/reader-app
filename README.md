# StoryNest Reader App

Мобильное приложение для чтения книг и прослушивания аудиокниг. Проект собран на Expo и React Native, поддерживает импорт пользовательских файлов, библиотеку с полками, экран чтения, аудиоплеер, заметки и настройки отображения текста.

## Возможности

- Импорт книг в форматах EPUB, FB2, TXT, PDF и ZIP.
- Извлечение названия, автора и обложки из EPUB/FB2, если эти данные есть в файле.
- Чтение EPUB, FB2, TXT и текстовых файлов внутри ZIP-архивов.
- Импорт аудиокниг отдельным аудиофайлом или ZIP-архивом с главами.
- Поддержка аудиотреков MP3, M4A, M4B, AAC, WAV, OGG, FLAC, OPUS и других распространённых форматов.
- Аудиоплеер с прогрессом, перемоткой, переходом между главами и синхронизацией с текстом.
- Полки: последние книги, все книги и пользовательские категории.
- Карточка книги с вкладками `Книга` и `Аудио`.
- Экран чтения с темами, настройкой шрифта, яркости, режима перелистывания, поиском, оглавлением, закладками и заметками.
- Локальное сохранение импортированных книг и прогресса чтения.

## Структура проекта

```text
app/                    Роуты Expo Router и корневой layout
components/             Экраны и UI-компоненты приложения
components/audio/       Аудиоплеер и привязка аудио к книге
components/bookInfo/    Экран информации о книге и редактирование
components/reader/      Экран чтения книги
components/shelfDetail/ Экран выбранной полки
shared/                 Общие типы, импорт, парсинг книг, аудиологика, хранение
assets/                 Шрифты, иконки, изображения и слои 3D-книги
```

## Запуск

Установите зависимости:

```bash
npm install
```

Запустите Expo:

```bash
npm start
```

Или отдельно для платформ:

```bash
npm run android
npm run ios
npm run web
```

## Проверка проекта

Проверка TypeScript:

```bash
npx tsc --noEmit
```

Проверка lint:

```bash
npm run lint
```

## Важные замечания

- PDF-файлы можно импортировать, но полноценное чтение PDF внутри приложения пока не реализовано.
- Пользовательские книги и прогресс сохраняются локально на устройстве.
- Для web-версии используется `localStorage`, для мобильной версии - файловое хранилище Expo.
- Backend в текущей версии не используется.

## Статус

Проект находится в стадии MVP и готов для демонстрации основного пользовательского сценария: импорт книги, чтение, создание заметки, работа с полками и прослушивание аудио.
